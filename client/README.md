```js

  globalIgnores(['dist']),

      // Optionally, add this for stylistic rules

      // Other configs...

      parserOptions: {
        project: ['./tsconfig.node.json', './tsconfig.app.json'],
        tsconfigRootDir: import.meta.dirname,
      },
      // other options...
    },
  },
])
```

You can also install [eslint-plugin-react-x](https://github.com/Rel1cx/eslint-react/tree/main/packages/plugins/eslint-plugin-react-x) and [eslint-plugin-react-dom](https://github.com/Rel1cx/eslint-react/tree/main/packages/plugins/eslint-plugin-react-dom) for React-specific lint rules:

# StemSplitter Full Stack App

This project consists of a Flask backend for audio stem separation and a React frontend for uploading audio and downloading separated stems.

## Getting Started

### 1. Backend (Flask + Demucs)

```
cd server
pip install -r requirements.txt
python audio_api.py
```
The server will run at http://localhost:5000

### 2. Frontend (React + Vite)

```
cd client
npm install
npm run dev
```
The app will run at http://localhost:5173

## Usage
- Open the frontend in your browser.
- Upload an audio file (mp3, wav, etc.).
- Click "Upload & Separate".
- Download the resulting zip file with separated stems.

## Notes
- The backend currently uses a placeholder for stem separation. Integrate Demucs in `server/audio_processing.py` for real separation.
- Make sure CORS is handled if accessing the backend from a different host/port during development.
